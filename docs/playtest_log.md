# プレイテストログ

## 2026-05-25: Prototype 0 ベースライン

実装済み:

- ローカル人間プレイヤーと簡易ボット。
- プロシージャルなセクター生成。
- 遅延後にグラフ伝播する感染ノード。
- 感染ノード設置中の伝播プレビュー。
- 連鎖起動。
- 一時ファイアウォール。
- ゲート、データストリーム、不安定セクター。
- コンパクトなサイバーHUD。

初回ハンズオンで確認する問い:

- パケット経路はヒット前に読めるか。
- プレビューによって読み合いが明確になりすぎていないか。
- ボットは回避テストに十分な圧力を作れているか。
- 一時ファイアウォールは有用か、それとも反応的すぎるか。
- 伝播は古典的な爆風ではなく、サイバー戦に見えるか。
- 起動時間は予測に十分長く、圧力を保つ程度には短いか。

初期チューニング仮説:

- 撃破が理不尽なら、伝播を15〜25%遅くするか、予兆パルスを強くする。
- プレイヤーが停滞するなら、ファイアウォール持続時間を短くするか、ノード容量を増やす。
- 連鎖反応が偶然に見えるなら、連鎖起動時のノード警告アークを明確にする。
- マップが詰まりすぎるなら、ソフトファイアウォール密度より先にハードファイアウォール密度を下げる。
