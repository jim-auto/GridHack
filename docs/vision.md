# ビジョン

## ワーキングタイトル

GridHack

## コアファンタジー

プレイヤーは、崩壊中のメガコーポ・ネットワークに侵入したAIです。すべての移動は、帯域、予測、そして生存可能な空間を奪い合う判断になります。

## デザインの柱

1. **爆発ではなく、ネットワーク伝播**  
   感染ノードは接続されたネットワーク経路に沿って汚染を放出します。見た目は爆風ではなく、パケットストーム、信号の分岐、インフラ侵食として読めるべきです。

2. **攻撃的な空間制圧**  
   最適解は安全地帯で待つことではありません。経路を断ち、回避を強要し、連鎖反応を仕込み、危険な領域へ適切なタイミングで踏み込むプレイが強くなるべきです。

3. **読みやすいサイバー抽象表現**  
   キャラクター、脅威、地形は、トップダウン視点で角張り、幾何学的で、高コントラストに見える必要があります。マスコット体型、玩具的な爆弾、レトロなアリーナの記号は使いません。

4. **現代的なテンポ**  
   移動は滑らかにし、判断時間は短めにし、伝播はインフラを曲がりながら進みます。戦術ツールは単純な設置だけでなく、切り返しを生むものにします。

## 古典的な爆弾アリーナとの差別化

- 固定の十字爆風ではなく、生成されたネットワークリンク、ゲート、不安定ノード、データストリームを伝播が進みます。
- 防御手段は静的な籠もりではなく、一時ファイアウォールと経路ハックです。
- プロトコル拡張は単純な強化アイテムではなく、戦術的なロードアウト変異です。
- アリーナは同一構造の迷路ではなく、サイバーインフラのレイアウトです。
- 視覚言語はダークサイバーパンク、ホログラム、グリッチ、抽象表現です。

## 初期検証の問い

プレイヤーは汚染がどこへ進むかを素早く理解でき、同時に連鎖反応やインフラ相互作用に驚きを感じられるか。
