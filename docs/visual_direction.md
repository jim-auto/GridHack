# ビジュアル方向性

## アートの約束

GridHack は、敵対的なインフラ内部で行われる戦術的サイバー戦として見えるべきです。暗い基盤、ネオンのパケット流、角張ったAIシルエット、ホログラフィックUIを中心にします。

## パレット

ベース:

- ニアブラック: `#03040A`
- ディープネットワークブルー: `#07111D`
- パネルグラファイト: `#111827`

主要ネオン:

- シアン: `#00F2FF`
- マゼンタ: `#FF1493`
- バイオレット: `#B238FF`

補助色:

- 警告レッド: `#FF355E`
- シグナルグリーン: `#39FF88`
- クールホワイト: `#DDEBFF`

シアンとマゼンタはプレイヤーと脅威の可読性に使います。画面全体を単一色相で埋めず、危険色が意味を持つように、プレイフィールドは暗いニュートラル構造を保ちます。

## 形状言語

プレイヤー:

- 角張った六角形系シルエット。
- 向きを示す非対称フィンまたはアンテナ。
- 中空のホログラムコア。
- 丸いマスコット頭部、手袋、靴、かわいい比率は禁止。

感染ノード:

- 不規則な五角形のハードウェア片。
- 周回する進捗アーク。
- 小さなパケットのちらつき線。
- 球状爆弾や導火線アニメーションにはしない。

ファイアウォール:

- 密度のあるスラブ、セキュリティバー、斜めのロック構造。
- 永続ファイアウォールは冷たいシアンブルー。
- 破壊可能ファイアウォールはマゼンタ汚染。
- 一時ファイアウォールは所有者色を継承し、投影物に見せる。

伝播:

- リンクに沿ったパケットパルス。
- 分岐する線分。
- 短命の信号ヘッドと残光。
- 固定の四方向十字や丸い爆発雲は避ける。

## シルエット案

1. **Null Vector**  
   細い六角形ボディ、前方ブレード、空洞コア。速く精密な印象。

2. **Red Ghost**  
   分割された三角シェル、後方フィン、ずれたコア。攻撃的で不安定。

3. **Latch Seraph**  
   ダイヤ形ボディと左右のフック。捕獲プログラムのように見える。

4. **Shard Monk**  
   縦長の凧形シルエットと最小限の翼。防御的で読みやすい。

5. **Index Wraith**  
   幅広のステルス三角形と欠けた後端。視界妨害テスト向け。

## 探索するサイバースタイル

- コーポレート・セキュリティ層: 無菌的なシアングリッド、厳密な幾何、清潔なロックアイコン。
- 汚染メモリ: バイオレットの走査線、壊れたピクセル、不安定な斜め運動。
- ブラックマーケット・プロトコル層: マゼンタのデカール、違法グリフ、オーバークロックアーク。
- ディープカーネル・セクター: ほぼ単色の基盤に、まれな高輝度パルス。

## アセット制作ルール

AIコンセプトアートは参照用途のみです。最終ゲーム内アセットは、制御されたパレット、明快なシルエット、実プレイサイズでの可読性を持つトップダウン形状へ描き直すかモデリングします。
